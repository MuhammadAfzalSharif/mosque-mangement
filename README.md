## üìñ About The Project

The **Mosque Management System** is a dual-purpose application designed to serve both the Muslim community and mosque administrators. It solves the challenge of finding accurate, up-to-date prayer times and mosque information while providing a seamless management interface for mosque officials.

### üéØ Who is this for?

| User Type            | Benefits                                                                             |
| -------------------- | ------------------------------------------------------------------------------------ |
| **üë§ Public Users**  | Find nearby mosques, view real-time prayer clocks, and access facility details.      |
| **üïå Mosque Admins** | Manage prayer times, update mosque profiles, and communicate with the community.     |
| **‚ö° Super Admins**  | Oversee the platform, approve new mosque registrations, and manage system integrity. |

---

## üöÄ Key Features

### 1. Smart Mosque Finder

- **Search & Filter**: Locate mosques by name or location.
- **Live Prayer Clock**: Dynamic countdown to the next prayer.
- **Detailed Profiles**: View facilities, photos, and contact info.

### 2. Admin Dashboard

- **Secure Login**: Protected access for mosque administrators.
- **Real-time Updates**: Instantly update prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha, Jummah).
- **Profile Management**: Edit mosque descriptions and location details.

### 3. Super Admin Control

- **Registration Workflow**: Review and approve/reject mosque applications.
- **Audit Logs**: Track all system changes and updates.
- **User Management**: Manage admin accounts and permissions.

## üõ†Ô∏è How It Works

1.  **Registration**: A mosque representative registers their mosque.
2.  **Verification**: The Super Admin reviews the application and approves it.
3.  **Management**: The Mosque Admin logs in to set prayer times and details.
4.  **Discovery**: Public users search and find the mosque with the latest data.
